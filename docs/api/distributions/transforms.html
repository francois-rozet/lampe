class lampe.distributions.transforms.MonotonicAffineTransform(shift, scale, eps=0.001, **kwargs)#

Creates a transformation \(f(x) = x \times \text{softplus}(\alpha) + \beta\).

Parameters

  • shift (Tensor) – The shift term \(\beta\), with shape \((*,)\).

  • scale (Tensor) – The unconstrained scale factor \(\alpha\), with shape \((*,)\).

  • eps (float) – A numerical stability term.

class lampe.distributions.transforms.MonotonicRationalQuadraticSplineTransform(widths, heights, derivatives, bound=3, eps=0.001, **kwargs)#

Creates a monotonic rational-quadratic spline transformation.

References

Neural Spline Flows (Durkan et al., 2019) https://arxiv.org/abs/1906.04032

Parameters

  • widths (Tensor) – The unconstrained bin widths, with shape \((*, K)\).

  • heights (Tensor) – The unconstrained bin heights, with shape \((*, K)\).

  • derivatives (Tensor) – The unconstrained knot derivatives, with shape \((*, K - 1)\).

  • bound (float) – The spline’s (co)domain bound \(B\).

  • eps (float) – A numerical stability term.

class lampe.distributions.transforms.MonotonicTransform(f, bound=10.0, eps=1e-06, **kwargs)#

Creates a transformation from a monotonic univariate function \(f(x)\).

The inverse function \(f^{-1}\) is approximated using the bisection method.

Wikipedia

https://wikipedia.org/wiki/Bisection_method

Parameters

  • f (Callable[[Tensor], Tensor]) – A monotonic univariate function \(f(x)\).

  • bound (float) – The domain bound \(B\).

  • eps (float) – The numerical tolerance for the inverse transformation.

Tranform via an autoregressive mapping.

Parameters

  • meta (Callable[[Tensor], Transform]) – A meta function \(g(x) = f\).

  • passes (int) – The number of passes for the inverse transformation.
