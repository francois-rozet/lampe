lampe.priors#

Priors and distributions.

class lampe.priors.BoxUniform(lower, upper, ndims=1)#

Creates a distribution for a multivariate random variable \(X\) distributed uniformly over an hypercube domain. Formally,

\[l_i \leq X_i < u_i ,\]

where \(l_i\) and \(u_i\) are respectively the lower and upper bounds of the domain in the \(i\)-th dimension.

Parameters

  • lower (Tensor) – The lower bounds (inclusive).

  • upper (Tensor) – The upper bounds (exclusive).

  • ndims (int) – The number of batch dimensions to interpret as event dimensions.

Example

>>> d = BoxUniform(-torch.ones(3), torch.ones(3))
>>> d.event_shape
torch.Size([3])
>>> d.sample()
tensor([ 0.1859, -0.9698,  0.0665])
class lampe.priors.DiagNormal(loc, scale, ndims=1)#

Creates a multivariate normal distribution parametrized by the variables mean \(\mu\) and standard deviation \(\sigma\), but assumes no correlation between the variables.

Parameters

  • loc (Tensor) – The mean \(\mu\) of the variables.

  • scale (Tensor) – The standard deviation \(\sigma\) of the variables.

  • ndims (int) – The number of batch dimensions to interpret as event dimensions.

Example

>>> d = DiagNormal(torch.zeros(3), torch.ones(3))
>>> d.event_shape
torch.Size([3])
>>> d.sample()
tensor([0.7304, -0.1976, -1.7591])

class lampe.priors.Sort(base, n=2, descending=False)#

Creates a distribution for a \(n\)-d random variable \(X\), whose elements \(X_i\) are \(n\) draws from a base distribution \(p(Y)\), ordered such that \(X_i \leq X_{i + 1}\).

\[\begin{split}p(X = x) = \begin{cases} n! \, \prod_{i = 1}^n p(Y = x_i) & \text{if $x$ is ordered} \\ 0 & \text{otherwise} \end{cases}\end{split}\]
Parameters

  • base (Distribution) – A base distribution \(p(Y)\).

  • n (int) – The number of draws \(n\).

  • descending (bool) – Whether the elements are sorted in descending order or not.

Example

>>> d = Sort(Normal(0, 1), 3)
>>> d.event_shape
torch.Size([3])
>>> d.sample()
tensor([-1.4434, -0.3861,  0.2439])
class lampe.priors.TopK(base, k=1, n=2, **kwargs)#

Creates a distribution for a \(k\)-d random variable \(X\), whose elements \(X_i\) are the top \(k\) among \(n\) draws from a base distribution \(p(Y)\), ordered such that \(X_i \leq X_{i + 1}\).

\[\begin{split}p(X = x) = \begin{cases} \frac{n!}{(n - k)!} \, \prod_{i = 1}^k p(Y = x_i) \, P(Y \geq x_k)^{n - k} & \text{if $x$ is ordered} \\ 0 & \text{otherwise} \end{cases}\end{split}\]
Parameters

  • base (Distribution) – A base distribution \(p(Y)\).

  • k (int) – The number of selected elements \(k\).

  • n (int) – The number of draws \(n\).

  • kwargs – Keyword arguments passed to Sort.

Example

>>> d = TopK(Normal(0, 1), 2, 3)
>>> d.event_shape
torch.Size([2])
>>> d.sample()
tensor([-0.2167,  0.6739])
class lampe.priors.Minimum(base, n=2)#

Creates a distribution for a scalar random variable \(X\), which is the minimum among \(n\) draws from a base distribution \(p(Y)\).

\[p(X = x) = n \, p(Y = x) \, P(Y \geq x)^{n - 1}\]
Parameters

  • base (Distribution) – A base distribution \(p(Y)\).

  • n (int) – The number of draws \(n\).

Example

>>> d = Minimum(Normal(0, 1), 3)
>>> d.event_shape
torch.Size([])
>>> d.sample()
tensor(-1.7552)

class lampe.priors.TransformedUniform(lower, upper, f)#

Creates a distribution for a random variable \(X\), whose transformation \(f(X)\) is uniformly distributed over the interval \([f(l), f(u)]\).

\[p(X = x) = f'(x) \frac{1}{f(u) - f(l)}\]
Parameters

  • lower (Tensor) – A lower bound \(l\) (inclusive).

  • upper (Tensor) – An upper bound \(u\) (exclusive).

  • f (Transform) – A transformation \(f\), monotonically increasing over \([l, u]\).

Example

>>> d = TransformedUniform(-1, 1, ExpTransform())
>>> d.event_shape
torch.Size([])
>>> d.sample()
tensor(0.5594)
